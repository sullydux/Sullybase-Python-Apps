# Sullybase Python Apps
## A collection of my smaller python apps.

Click the app name to go to the app download.

---

## Sullybase Clipboard Organizer

This is an application where users can input peices of data with a name and when they open the application they pick the name and hit copy. This alows users to not have to search for content but paste it to their clipboard easy and fast. The application lives in the menubar.

---

## Sullybase Filament Cost Finder

This is an application where users can record their filament messurments and cost. After putting filament information in users can enter how many grams or meters of filament they used and it will generate a cost for filament used. Users can also click the filament name and it will take them to the website they bought it from, if they put the link in the app. This makes it easy to price prints and re-buy filaments.

---

## Sullybase Notes

This is an application where users can take notes on two notepads. Open from the menubar and it will not change your current window so if you are on Google fullscreen, for example, it will keep you there with just a popup window. It also says the amount of words and characters used in each notepad.

---

## Sullybase Password Analyzer

This is an application where users can input passwords and it will rate them on different things including patterns, dictionary revelence, and entropy. It also has a quantum mode which is a harder grader with quantum formulas. It is a hard grader on current day passwords too. This app does not store or send passwords over the internet. It sends password snipits to a dictionary API to comferm words but never will send full passwords over. This has an on device, privacy first design.

---

## Time To Date

This is an application where users can set a time and date and have a countdown or countup in their menubar there is also a alarm option which alarms users at T+0 seconds. its easy to use.
